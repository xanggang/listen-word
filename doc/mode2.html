实现原理

核心CSS代码：

.shape {
    width: 400px;
    height: 200px;
    border-radius: 0;
    transition:
        width 0.6s cubic-bezier(0.68, -0.55, 0.27, 1.55) 0s,
        height 0.6s cubic-bezier(0.68, -0.55, 0.27, 1.55) 0.6s,
        border-radius 0.4s ease 1.2s,
        background 0.4s ease 1.2s;
}

.shape.collapsed {
    width: 60px;
    height: 60px;
    border-radius: 50%;
}

关键属性：

  • 宽度过渡：0.6秒，使用弹性效果，立即开始
  • 高度过渡：0.6秒，使用弹性效果，延迟0.6秒开始
  • 圆角过渡：0.4秒，使用缓动效果，延迟1.2秒开始
  • 背景过渡：0.4秒，使用缓动效果，延迟1.2秒开始

技术要点： 通过为每个属性设置不同的过渡时间和延迟，我们实现了分阶段的过渡效果。宽度首先变化，然后是高度，最后是圆角和背景色，这样过渡更加自然流畅。
